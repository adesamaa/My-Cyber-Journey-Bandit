# Levels 12-19: Text Processing & Data Extraction 
These levels focused on extracting and manipulating data from different file types.

### Key Levels & Takeaways

**Level 12-13: Strings in Binary Files**
- Used `strings` command to find readable text inside binary/data files
- Piped output with `grep` to filter results
- **Learned**: Binary files can contain hidden readable information
- **Real-world use**: Searching logs or memory dumps for suspicious strings in SOC work

**Level 14-16: Decoding & Hexdump**
- Worked with base64 decoding (`base64 -d`)
- Used `hexdump` to view file contents in hexadecimal
- **Learned**: Different ways data can be encoded or stored

**Level 17-19: Compression & Archives**
- Handled gzip, bzip2, tar, and zip files
- Combined multiple commands using pipes (`|`) and redirection (`>`)
- **Learned**: Power of chaining commands for complex tasks

**Overall from 12-19**:
Improved my ability to process text and extract useful information quickly — a core skill for log analysis and incident response.
