# Levels 20-24: Advanced Concepts (SSH Keys, Git, Cron, Automation)
These later levels introduced more real-world system administration concepts.

### Key Levels & Takeaways

**Level 20-22: SSH Keys**
- Generated and used SSH key pairs instead of passwords (`ssh-keygen`)
- Copied public key to authorized_keys
- **Learned**: More secure way to access remote servers
- **Real-world use**: Common practice in DevOps and IT Support environments

**Level 23-26: Restricted Shells & Escaping**
- Worked with limited shells and found ways to run commands
- Used editors like `vim` or other tricks to escape restrictions
- **Learned**: How shells can be restricted and basic bypass techniques

**Level 27-30: Git, Cron & Setuid**
- Used `git show` to view previous commits
- Learned about cron jobs for scheduled tasks
- Explored setuid binaries and their privilege implications
- **Learned**: Version control basics and how scheduled tasks work

**Level 31-34: Final Challenges**
- Combined many skills: environment variables, shell escaping, and complex command chaining
- **Learned**: Thinking creatively and putting everything together

**Overall from 20-34**:
Gained exposure to concepts used in real Linux administration, automation, and basic security awareness.
