# OverTheWire Bandit Walkthrough - Completed all the levels.

# INTRO 
I recently completed all the levels of the OverTheWire Bandit, a beginner-friendly wargame designed for newbies to learn linux command-line fundamentals in a fun way.
This repository documents the key commands used, and the most important concepts i learned. This game significantly improved my comfort with the terminal and i gained valuable skills directly applicable to SOC analysis, log review, and system troubleshooting.

#KEY TAKEAWAYS
- I astered core linux commands: ssh, ls, grep, cd, zip, tar, chmod, find, file, etc.
- I learned file and directory manipulation, permission management and simple bash scripting.
- as I advanced in levels, i built a problem-solving mindset through debugging, experimenting and reading man pages.
