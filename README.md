# OverTheWire Bandit - Completed All Levels

![Terminal](https://img.shields.io/badge/Linux-CLI-blue)  
**March 2026**

I recently completed **all 34 levels** of OverTheWire Bandit — a beginner-friendly Linux wargame that teaches essential command-line skills through practical challenges.

This repository serves as a record of my journey. It documents the **key commands**, **concepts learned**, and how these skills apply to real-world cybersecurity and IT roles (especially SOC analysis, log review, and system troubleshooting).

### Key Takeaways
- Mastered core Linux commands: `ssh`, `ls`, `cat`, `grep`, `find`, `awk`, `sed`, `chmod`, `tar`, `zip`, `file`, and more.
- Gained strong skills in file & directory manipulation, permission management, text processing with pipes/redirection, and basic Bash concepts.
- Built a practical problem-solving mindset by reading `man` pages, experimenting, debugging, and iterating on solutions.
- Improved terminal comfort significantly — skills I now apply daily in TryHackMe SOC Level 1 rooms and small Python scripting projects.
